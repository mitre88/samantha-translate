# Samantha Translate

부제: 실시간 음성 번역기

홍보 문구:
외국어를 듣고 내가 선택한 언어로 말해 주는 번역을 받아보세요.

설명:
Samantha Translate는 대화, 여행, 면접, 일상 청취를 위한 간단하고 프리미엄한 음성 번역 앱입니다.

앱을 열고 출력 언어를 선택한 뒤 듣기를 시작하면 Samantha가 실시간으로 번역을 음성으로 재생합니다. 앱은 원본 언어를 자동으로 감지하므로 대화 전마다 언어 조합을 설정할 필요가 없습니다.

개인정보 보호는 명확합니다. Samantha Translate는 오디오, 전사 내용 또는 번역 기록을 저장하지 않습니다. 오디오는 번역 제공을 위해 실시간으로 처리됩니다.

구독:
신규 구독자는 3일 무료 체험으로 시작할 수 있습니다. 체험 후 Samantha Translate는 갱신 24시간 전까지 취소하지 않는 한 매주 US$4.99로 자동 갱신됩니다. Apple 계정 설정에서 구독을 관리하거나 취소할 수 있습니다.
