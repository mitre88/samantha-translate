# Samantha Translate

Sous-titre : Traducteur vocal en temps réel

Samantha Translate écoute une langue et parle la traduction dans la langue choisie. L’app détecte automatiquement la langue source et ne stocke ni audio, ni transcription, ni historique de traduction.

Essai gratuit de 3 jours pour les nouveaux abonnés, puis 4,99 USD par semaine avec renouvellement automatique via Apple.
